# PendulumDynamics
This is a library used to simulate the motion of a parametrically driven pendulum with external forcing.
### Tested on:
 - MacOS 10.15.4
 - Ubuntu 20.04 LTS
 
 ### Requirements
  - [BOOST](https://www.boost.org/)
  - [HDF5](https://www.hdfgroup.org/)
  - [OpenMPI](https://www.open-mpi.org/)
